Welcome to Amazing Mahjongg 3D!

Congratulations on choosing Amazing Mahjongg 3D! It is a traditional Chinese game for centuries, Mahjongg combines elements of skill, strategy, memory, and intuition into a challenging game of patience. This is a single-player 3D version of a classic favorite.

Main features

  • True 3D game board
  • Different layouts
  • Different tile sets and ability to use your own tile sets
  • Beautiful background images
  • Realistic lighting
  • Sound

System requirements

  • Windows 98/ME/2000/XP
  • 400 MHz Pentium II or higher
  • 64MB RAM
  • 8 MB 3D video card
  • 16-bit color or better
  • Microsoft-compatible Mouse
  • 5 MB HD space
