Wie gestalte ich meine eigenen Tilesets

Es ist sehr einfach eigene Spielsteine für Amazing Mahjongg 3D zu machen. Zuerst müssen Sie eigene Bilder entwerfen oder kopieren. Jedes Bild muss als eigene Datei abgespeichert werden. Es ist besser (aber nicht Voraussetzung) wenn alle Bilder die gleiche Größe haben. Die Spielsteine werden am Besten dargestellt wenn die Breite 75% der Höhe beträgt (z.B. 100 Pixel Höhe auf 75 Pixel Breite). Die Dateien müssen als .JPG abgespeichert werden.

Danach sollten Sie die Dateien unter den folgenden Namen abspeichern:

  • Ein leeres Feld, welches für die Seiten und Rücken benutzt wird - benannt 00.jpg
  • Die Spielsteine der Kategorie "Kreise" - benannt von 01.jpg bis 09.jpg
  • Die Spielsteine der Kategorie "Bambus" - benannt von 11.jpg bis 19.jpg
  • Die Spielsteine der Kategorie "Charaktäre" - benannt von 21.jpg bis 29.jpg
  • Die Spielsteine der Kategorie "Winde" - benannt von 31.jpg bis 34.jpg (Nord, Süd, Ost, West)
  • Die Spielsteine der Kategorie "Drachen" - benannt von 10.jpg für Weiß, 20.jpg für Grün und 30.jpg für Rot.
  • Die Spielsteine der Kategorie "Jahreszeiten" - benannt von 35.jpg, 36.jpg, 37.jpg, 38.jpg
  • Die Spielsteine der Kategorie "Blumen" - benannt von 39.jpg, 40.jpg, 41.jpg, 42.jpg

Jetzt erhalten Sie ein neues Tileset mit 43 Steinen mit den Dateien 00.jpg bis 42.jpg.

Wenn Sie zusätzlichen Informationen oder Ihren Namen dem Tileset beifügen möchten, so müssen Sie eine Textdatei mit Namen tileset.ini (z.B. mit dem Notepad.exe) hinzufügen.

Die Textdatei muss mehrere festgelegte Zeile enthalten. Wichtig ist der Name des Tilesets: "Name = XYZ" und der Autor: "Author = XYZ". Als Beispiel - Ihre tileset.ini könnte so aussehen
Name=Mein erstes eigenes Tileset
Author=Michael Mustermann

Als letztes müssen diese Dateien komprimiert werden (00.jpg - 42.jpg und tileset.ini). Dies kann mit jedem .ZIP Programm (z.B. WinZip, PkZip, usw.) geschehen. Man kann die Endung des gepackten Files als .zip belassen oder zu .mjt ändern.

Jetzt ist das neue Tileset (ein kompleter Satz neuer Spielsteine) fertig. Kopieren Sie die Datei noch in das Amazing Mahjongg\Tile sets Verzeichnis und Sie können danach gleich loslegen.
